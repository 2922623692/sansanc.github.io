随机数（random）

2020-03-23
生成0-10的随机数
import java.util.Random;
class a
{
    public static void main(String[] args)
    {
        Random random=new random();
        int a=Math.abs(random.nextInt()%10);
        system.out.println(a);
    }
}

生成5-10的随机数
import java.util.Random;
class a
{
    public static void main(String[] args)
    {
        Random random=new Random();
        int a=Math.abs(random.nextInt()%6+5);
    }
}
