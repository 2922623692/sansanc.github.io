C#Textbox

2020-03-21
  • 让文本允许多行 MultiLine属性设置为true即可
  • 让文本右侧出现滚动滑块scrollBars属性 设置为Both（超出时显示）vertical（只显示垂直滑块，只有设置Dock属性后才会出现横向滑块）
  • 需要文本只读，设置readonly为true
    • 与enable的区别
      • readonly允许复制内容，enadle不允许
      • vs可以直观看见编程灰色，en只有运行后才可以看见

事件

  • textChanged事件获取的是文本框的内容，并且每次按钮触发的单独内容。
string str=text1.text;
text2.text=str+"\r\n"//第二个文本框获得第一个1内容（文本等于输入的）

string str=text1.text;
text2.AppendText(str);//追加字符串

string str=text1.text;
text2.AppendText(str+"长度"+text1.TextLength.ToString()+"\r\n")
  • TextChanged事件获取的是文本框的所有内容，并非每次按键触发的单独内容。

    • AppendText的方法：追加文本，在末尾追加新的数据

    • TextLength当前文本内容的长度（个数）
