java super关键字

2020-03-21
  • 1.在子类成员方法中，访问父类的成员变量。
  • 2.在子类的成员方法中访问父类的成员方法。
package Textsuper;
/**
 * super关键字的使用
 * @author 蒲国林
 *super关键字的用法有三种：
 *1.在子类成员方法中，访问父类的成员变量。
 *2.在子类的成员方法中访问父类的成员方法。
 */
public class test {
	public static void main(String[] args) {
		Zi z=new Zi();
		z.A();
	}
}
