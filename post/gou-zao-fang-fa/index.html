java 构造方法

2020-03-21
修饰符列表 构造方法名(形式参数列表)
{
    构造方法体;
}

注:
对于构造方法来说，返回值类型不休要指定并且也不能写void，只要写了void，那么这个方法就成为普通方法

构造方法作业：通过构造方法调用，可以创建对象

调用：

new 构造方法(实参列表);

注：构造方法支持方法重载

this关键字

  1. this关键字翻译为 这个
  2. this是一个引用，this是一个变量，this变量中保存了内存的地址指向本身，this存储在java堆内存中java对象内部
  3. 创建多个对象就有多个this
  4. this出现在实例方法中，this表示指向当前对象执行这个对象的对象（this代表当前对象）
