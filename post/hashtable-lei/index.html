HashTable类

2020-03-21
  • OHashTable翻译为哈希表，表示键( key ) /值( value )对的集合。
    • OHashTable像一个字典,根据键可以找到相应的值。
    • 口哈希表提供了添加元素和访问元素等方法,哈希表，名-值对。OGetHashCode(方法返回一个int型数据，使用这个键的值生成该int型数据。
    • 口哈希表获取这个值最后返回一个索引,表示带有给定散列的数据项在字典中存储的位置。
    • 口常见的哈希表操作有:构造哈希表、删除哈希表元素和遍历哈希表。
