html表单元素

2020-03-21

imput：输入表单元素

<inputt type="属性值" />

type:
button 定义点击按钮
checkbox 定义复选框
file 定义输入字段和浏览按钮提供文件上传和下载
hidden 定义隐藏的输入字段
image 定义图像形式的提交按钮
password 定义密码字段
radio 定义单选按钮
reset 定义重置按钮
submit 定义提交按钮
text 定义单行输入字段，文本
range 滑动输入类型
number 手动输入值包含range属性1
input：

    Email <input type="email"> 验证电子邮件地址
    Tel<input type="tel">验证电话号码
    url<input type="url">验证url完整度

input元素获取时间

   <input type="date">

input元素获取颜色

    <input type="color">

input元素获取搜索用词

    <input type="search">

input元素获得隐藏数据

    <input type="hidden">

input元素生成图像按钮

    <input type="image" src="图像地址" width="长">

input上传文件

    <input type="file" Multiple(允许上传多个文件)/Required(允许上传一个文件)>

value与placeholder区别

    value="123" 文本框字符实例化
    placeholder="123" 文本框不实例化
    maxlength（text）
    maxlength="8"规定文本框最大可输入字符长度
    size（text）
    size="20"拓宽单行文本框
    readonly（text）
    readonly 只读

<datalist>（text）
<input typr="text" list="A">
<datalist>
<option>内容</option>
</datalist>
