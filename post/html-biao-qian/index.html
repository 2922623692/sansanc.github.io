标题标签：

<h1>—<h6>

段落标签:

<p></p>

换行标签：

<br></br>

文本格式化：

<stong></stong> 加粗 或者 <b></b>
<em><em> 倾斜]
<del></del> 删除线
<ins></ins> 下划线

和标签:

<div></div> 分割，分区
<span></sapn>  跨行，跨度

####图像标签：

<img src="图像url">

* alt 替换文本，图像显示不出来时用文字替代
* title 将鼠标放在个上，显示文字
* boede 给图像设置边框
创建分区响应图:
rect代表一个矩形 四个逗号组成 分别是左上右下
circle 代表一个圆形 上逗号 圆心的x 圆心的y  半径
poly 多边形 至少六个逗号 
dafault 默认整张图片

<img src="图片地址" width="长度" usemap="#定义名字">
	<map name="定义的名字">
		<area herf="跳转地址" shape="rect/circle/poly/dafault" coords="" harget="_blank">
	</map>

超链接标签：

<a href="跳转路劲" taget="窗口弹出方式"></a>
  • taget：_self 默认值 _blank在新窗口中打开
  • 分类：多部，内部，空，下载链接

锚点标签：

<a href="#two">A</a>
<h3 id="two">B</h3>

表格标签：

<table><tr>...</tr></table>

* <rable></rable> 定义表格标签
* <tr></tr> 定义表格行
* <td></td> 定义表格列
* <th></th> 标题单元格 居中加粗

表格属性：

* broder("1"有边框，" "没有边框)
* cellpading（默认为1像素）
* cellspacing（规定单元格之间的距离）
* width（规定表格宽度）

表格结构标签：

* <thead></thead>定义表格头部区域
* <tbody></tbody>定义表格主体

合并单元格:

rowspan 跨行合并
colspan 跨列合并
