CSS过度

2020-03-21
：hover{
transition-delay：150ms；延迟多长时间
transition-duration：500ms；持续多长时间
transitiion-property：width；宽度瞬间变化
transition-timing-function：
easet 逐渐加速
ease-in 由慢到快
ease-out 由快到慢
ease-in-out 前面和后面快 中间快

}

例如：

		p {
			width:100px;
			height: 100px;
			background-color: antiquewhite;
		}
		p:hover {
			width: 150px;
			height: 150px;
			background-color: red;
			transition-delay: 150ms;
			transition-duration: 500ms;
		}
