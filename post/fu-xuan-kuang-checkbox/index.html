复选框Checkbox

2020-03-21
checked属性，标记当前复选框状态（是否被选中）
ThreeState 会提供三种选中状态
  • unchecked没被选中
  • checked被选中
  • indeterminate[树结构时，会出现一种状态，表示当前树结构下有一个或者多个（并非所有子项）子项被选中]

事件CheckedChanged

  • 指示当前控件是否被选中
private void chk1_CheckedChanged(object sender,EventArgs e)
{
    if(chk1.checked)//checked 当前是否被选中
    //chk1.checked 复选框1状态
    {
        MassageBox.show("被选中");
    }
    else
    {
        MassageBox.show("没有被选中");
    }
}
