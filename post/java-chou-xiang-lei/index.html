java抽象类

2020-03-21
  • 抽象方法就是加上abstract关键早，然后去掉大括号，直接分号结束
  • 抽象类：抽象方法所在的类必须是抽象类 定义：在class前加abstract
  • 抽象类的使用和定义：
  • 1.不能直接new抽象类对象
  • 2.必须一个子类来继承抽象父类
  • 覆盖重写（实现）:子类去掉抽象方法的abstract然后加上大括号
package TextAbstract;
/*
* 抽象方法就是加上abstract关键早，然后去掉大括号，直接分号结束
* 抽象类：抽象方法所在的类必须是抽象类 定义：在class前加abstract
* 
* 抽象类的使用和定义：
* 1.不能直接new抽象类对象
* 2.必须一个子类来继承抽象父类
* 覆盖重写（实现）:子类去掉抽象方法的abstract然后加上大括号
*/
public abstract class Animal {
   //抽象方法的定义
   public abstract void eat();
   //普通的成员方法
   public void A(){
   	
   }

}

package TextAbstract;

public class Cat extends Animal{

   @Override
   public void eat() {
   	System.out.println("猫吃鱼");
   }

}
